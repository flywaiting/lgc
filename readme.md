### 程序结构

## 踩坑
- ajax 轮询
   - status不能乱用...
- sqlite 查询
   - 避免关键字
- embed 目录限制
   - `fs.Sub(embed.FS, "root")`, 去除`root`路径
   - 只能包含当前目录, 及其子目录内容
- context 管理
- `time.Time` json 转换
- `sql tag`的实现
- `golang`条件编译, 交叉编译

## vue
- 数据绑定
- class 动态变化

## 行为

1. 添加任务
   1. 将任务插入数据库
2. 执行任务
   1. 从数据库取出任务相关数据
   2. 创建任务数据结构, 填充初始数据
   3. 运行任务
3. 结束任务
   1. 结束任务
   2. 更新数据库任务相关信息
4. 开始下一个任务
